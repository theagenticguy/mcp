# Login Screen Customization with shadcn/ui

This guide shows how to create a themed login screen using Amplify Authenticator and shadcn. The login screen should be a 50/50 split with the left side showing an image placeholder. and right side with the amplify authenticator v/h centered. 

## create a cutom Amplify theme (/src/lib/amplify-theme.ts) that matches the shadcn theme

## update main.tsx to include the theme

```tsx

<StrictMode>
    <ThemeProvider theme={amplifyTheme}>
      <App />
    </ThemeProvider>
  </StrictMode>

  ```

## adding brand images

Use the nova canvas MCP server if available to create a 1024x1024 splash image for the login page. This should be a minimal image that is contextual to the purpose of the application.

Also create a short modern name for the application and update the app name in the header
